# 資料集清洗專案

本專案用於整理與清洗來自不同事實查核平台的新聞資料集。

##  目錄結構

### MyGoPen
MyGoPen 事實查核平台的資料集處理

- **0_整合前資料集&最初抓取資料筆數進行比對** - 原始資料集與初始爬取資料的比對
- **1_清洗** - 使用整合後的資料進行清洗作業
  - 資料範圍：2020/01/01 ~ 2025/06/08
  - 主要檔案：`mygopen整合最終待清洗版(20200101~20250608).xlsx`

### TFC
台灣事實查核中心 (TFC) 的資料集處理

包含三個不同版本的資料處理結果：
- **20250905_072748_錯誤類別** - 第一版錯誤類別處理
- **20250905_104852_部分錯誤類別** - 部分錯誤類別處理
- **20250905_190228_錯誤 x 部分錯誤類別** - 錯誤與部分錯誤類別交叉處理

### Cofacts
Cofacts 真的假的平台資料集處理

##  資料格式說明

- 統一使用 **XLSX** 格式進行資料清洗
- CSV 檔案已移除，以確保格式一致性

##  資料處理流程

1. **資料爬取** - 從各平台爬取原始資料
2. **資料整合** - 整合不同來源的資料集
3. **資料清洗** - 進行資料清理與標準化
4. **資料驗證** - 確認資料品質與完整性
